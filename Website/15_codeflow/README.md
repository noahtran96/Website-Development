# 🚀 CodeFlow AI Landing Page

A modern **AI-powered development platform** landing page built with **React + TypeScript + TailwindCSS**. This project showcases intelligent coding tools, automated workflows, and premium pricing plans with a sleek, dark-themed interface.

---

## 🌐 Deployment

👉 https://15codeflow.vercel.app/

---

## 🛠️ Tech Stack

- ⚛️ **React + TypeScript**
- 🎨 **TailwindCSS** (Custom animations & Glassmorphism)
- 📦 **React Syntax Highlighter** (Night Owl theme for IDE simulation)
- 💅 **Lucide React** (Modern UI icons)
- 🔌 **Simple Icons** (Social media icons)

---

## ✨ Features

- **Interactive IDE Showcase:** A dynamic Hero section that allows users to toggle between different code examples (`App.jsx`, `Hero.jsx`, `Navbar.jsx`) with floating AI status cards.
- **Mouse-Tracking Glow:** Custom radial-gradient effect that follows the user's cursor for a premium feel.
- **AI Feature Roadmap:** Detailed sections for AI Code Completion, Automated Testing, and Smart Debugging using alternating layouts.
- **Tiered Pricing:** Clean pricing tables (Starter, Professional, Enterprise) with a "Most Popular" badge and hover glow effects.
- **Social Proof:** Testimonials section featuring developer feedback with high-quality avatars.
- **Fully Responsive:** Mobile-first approach with a functional mobile menu (hamburger toggle).
- **Scroll-Aware UI:** Navbar transitions from transparent to blurred glass effect upon scrolling.

---

## ⚙️ Getting Started

Install dependencies:  
`npm install`

Run the project:  
`npm run dev`

The app runs locally at:  
`http://localhost:5173`
