# 🚀 EdgeAI Landing Page

A modern **AI SaaS landing page** built with **React + TypeScript + TailwindCSS**, showcasing AI services, pricing plans, and business solutions with a clean, responsive design.

---

## 🌐 Deployment

👉 https://14saaslp.vercel.app/

---

## 🛠️ Tech Stack

- ⚛️ React + TypeScript
- 🎨 TailwindCSS (custom theme + dark mode)
- 🧠 Zustand (state management)
- 💅 Responsive Design
- 🌙 Dark / Light Mode Toggle

---

## ✨ Features

- Responsive landing page design
- Dark / Light mode toggle (Zustand + persist)
- Clean reusable component structure
- Smooth scrolling navigation
- Modern UI with TailwindCSS
- AI SaaS-focused layout
  - Hero section (headline + email CTA)
  - Brand logos (social proof)
  - Services section (AI capabilities)
  - About Us (mission & vision)
  - Pricing plans (Basic / Pro / Enterprise)
  - Call-to-action section

---

## ⚙️ Getting Started

Install dependencies:  
`npm install`

Run the project:  
`npm run dev`

The app runs locally at:  
`http://localhost:5173`
