# 💄 Skincare Landing Page (React + TypeScript)

A modern skincare landing page built with React and TypeScript as a learning project, inspired by an e-commerce style beauty website.
The application showcases skincare products in a clean, visually appealing layout with interactive sections, animations, and customer testimonials.

## ✨ Features

- Hero section with animated product showcase
- Responsive navigation bar
- Featured product slider using Swiper
- Product filtering by category:
  - All
  - Skincare
  - Conditioners
  - Foundations

- Interactive virtual try-on comparison section
- Testimonials carousel
- Footer with contact, company, and resource information
- Smooth UI animations with Framer Motion and Auto Animate
- Reusable React components with modular CSS styling

## 🛠 Tech Stack

- React
- TypeScript
- CSS Modules
- Framer Motion
- Swiper
- React Compare Image
- @formkit/auto-animate
- Heroicons
- React Icons

## 🚀 Getting Started

Install dependencies:  
`npm install`

Run the project:  
`npm run dev`

The app runs locally at:  
`http://localhost:5173`

## 📚 Learning Outcomes

- Understanding React component-based architecture with TypeScript
- Working with reusable UI sections and modular styling  
  Managing and filtering product data with state
- Implementing sliders and carousels using Swiper
- Adding animations with Framer Motion and Auto Animate
- Building interactive before/after comparison sections
- Structuring a responsive landing page for a product-based brand
